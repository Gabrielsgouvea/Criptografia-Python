# Criptografia-Python
# 🔐 Sistema de Criptografia ASCII

Atualização de um script original de 2012, migrado de Python 2 para **Python 3** com melhorias.

# 🚀 Funcionalidades

- **Codificar:** Transforma texto em códigos ASCII misturados com caracteres aleatórios ("fantasmas") e salva em `.txt`.
- **Decodificar:** Lê o arquivo `.txt`, remove o ruído e exibe a mensagem original na tela.
- **Simplicidade:** Interface via terminal, sem instalações complexas.
